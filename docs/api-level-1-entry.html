API Level 1: Entry Interfaces

← Back to overview

1) CLI Entry

  • File: src/main.py
  • Command: python -m src.main <case_folder>
  • Responsibility: initialize workspace, run scanner, coordinator, specialist pool, and synthesis in order.

2) Web Entry

  • File: app.py
  • Command: streamlit run app.py
  • Tabs: Run MDT, Debug workspace artifacts, Admin config editor.
  • Responsibility: user interaction only; execution still delegates to python -m src.main.

Entry-Level Contract

Input is a case directory path or uploaded files. Output is generated artifacts under that case's .mdt_workspace/.
