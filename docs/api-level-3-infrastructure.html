API Level 3: Infrastructure & Adapters

← Back to overview

Scanner API (src/scanner.py)

  • Scanner.scan(case_dir) returns a Manifest
  • Supports .md .txt .json .csv .pdf .docx .xlsx
  • Excludes .mdt_workspace from scanning

File Bus API (src/file_bus.py)

  • Creates workspace structure (init_workspace())
  • Typed save/load methods for manifest/index/dispatch/opinions/report
  • Acts as inter-agent file message bus

OpenCode Adapter API (src/cli_client.py)

  • OpenCodeClient.run(...) wraps opencode run
  • Handles timeouts, missing CLI errors, and non-zero exits
  • Writes per-agent error logs into workspace errors/

Configuration Surface

  • config/system.yaml controls model, timeout, max concurrency, specialist registry, and workflow flags.
