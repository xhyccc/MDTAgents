API Level 2: Workflow Orchestration

← Back to overview

Coordinator API (src/coordinator.py)

  • Coordinator.run_index(manifest) → writes 01_index.json
  • Coordinator.run_dispatch(index) → writes 02_dispatch.json
  • Coordinator.run_synthesis(index, opinions) → writes 05_mdt_report.md

Coordinator is the control-plane agent for classification, specialist dispatch, and final report synthesis.

Specialist Pool API (src/specialist_pool.py)

  • SpecialistPool.run_parallel(dispatch) → executes specialists in ThreadPoolExecutor
  • Consumes specialists_required from dispatch payload
  • Produces specialist opinions and writes 03_opinions/*.md
